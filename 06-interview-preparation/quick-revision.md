# AI Agents

### Definition

Software systems capable of reasoning, planning, using tools, and executing tasks toward defined objectives.

### Core Components

- Model
- Memory
- Planning
- Tools
- Execution

### Workflow

User Request → Context → Planning → Tool Usage → Execution → Response

### Key Interview Points

- Agent vs traditional automation
- Single Agent vs Multi-Agent
- Planning layer responsibility
- Tool interaction model

---

# MCP (Model Context Protocol)

### Definition

Protocol for connecting AI systems with external tools and contextual information.

### Components

- Host
- Client
- Server

### Workflow

AI Application → MCP Client → MCP Server → External Systems

### Key Interview Points

- Problem MCP solves
- Architecture components
- Integration benefits

# A2A (Agent-to-Agent)

### Definition

Communication model enabling distributed coordination across multiple agents.

### Core Concepts

- Coordination
- Delegation
- Aggregation

### Workflow

Coordinator → Distributed Execution → Aggregation → Response

### Key Interview Points

- Why multi-agent systems
- Communication patterns
- Coordination challenges

# RAG

### Definition

Retrieval-Augmented Generation combines retrieval systems with language models.

### Components

- Retriever
- Knowledge Source
- Model
- Generation Layer

### Benefits

- Better contextual accuracy
- Reduced hallucination risk
- Knowledge grounding
