# Retrieval-Augmented Generation (RAG)

Retrieval-Augmented Generation (RAG) is an AI application pattern that combines information retrieval with Large Language Models to provide responses using external knowledge sources.

RAG improves LLM applications by providing relevant context at generation time instead of relying only on the model's pre-trained knowledge.
