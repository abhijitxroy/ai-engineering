# AI Agents

AI Agents are software systems that use AI capabilities to understand goals, make decisions, use tools, and perform actions with limited human intervention.

This section focuses on designing autonomous AI systems that can reason, plan, coordinate, and execute tasks.

---

# Purpose

Understand:

- What AI Agents are
- Agent architecture and components
- Planning and reasoning approaches
- Agent orchestration patterns
- Agent execution workflows

AI Agents extend AI applications by adding:

- Goal-oriented behavior
- Planning capabilities
