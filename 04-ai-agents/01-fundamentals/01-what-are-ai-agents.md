# What Are AI Agents

AI Agents are software systems that use AI models to understand goals, make decisions, use tools, and perform actions to achieve objectives with limited human intervention.

Unlike traditional AI applications that only generate responses, agents can reason about tasks, plan steps, interact with external systems, and adapt based on outcomes.

# AI Application vs AI Agent

## AI Application

Focuses on providing AI-powered functionality through predefined workflows.

Examples:

- Chat assistants
- Search applications
- Document analysis systems

---

## AI Agent

Adds autonomous capabilities.

Examples:

- Goal understanding
- Planning
- Decision making
- Tool usage
- Action execution

---

# Core Agent Capabilities

## Goal Understanding

Agents interpret user objectives and determine required outcomes.

---

## Planning

Agents break complex goals into smaller tasks and define execution steps.

---

## Reasoning

Agents evaluate information, make decisions, and select appropriate actions.

---

## Tool Usage

Agents interact with external capabilities such as APIs, databases, and software systems.

---

## Learning and Adaptation

Agents improve behavior through feedback, memory, and experience.
