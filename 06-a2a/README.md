# Agent-to-Agent (A2A) Protocol

Agent-to-Agent (A2A) enables AI agents to communicate, collaborate, and coordinate tasks with other agents through standardized interaction patterns.

This section focuses on A2A fundamentals, architecture, agent communication, security, and practical usage patterns.
