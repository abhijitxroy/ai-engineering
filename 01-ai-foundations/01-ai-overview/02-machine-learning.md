# Machine Learning

Machine Learning (ML) is a branch of Artificial Intelligence that enables systems to learn patterns from data and make predictions or decisions without being explicitly programmed for every scenario.

Machine learning focuses on creating models that improve their performance through experience and data.

---

# Purpose

Understand:

- What Machine Learning means
- How ML systems learn from data
- Major learning approaches
- How ML connects to modern AI systems

---

# Core Concepts

## Learning From Data

Machine learning systems use:

- Data examples
- Algorithms
- Statistical patterns
- Model training processes

The goal is to learn relationships that can be used for prediction or decision making.

---

## Types of Machine Learning

### Supervised Learning

Learns from labeled examples.

Examples:

- Classification
- Prediction
- Detection systems

---

### Unsupervised Learning

Finds patterns from data without predefined labels.

Examples:

- Clustering
- Pattern discovery
- Data grouping

---

### Reinforcement Learning

Learns through interaction and feedback.

Concepts:

- Actions
- Rewards
- Environment feedback
